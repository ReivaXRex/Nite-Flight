# Nite-Flight
An owl's late night flight gets interrupted by a slew of planes.

Avoid the planes and build up your High Score!

----------------------------------------------------------------

### CONTROLS: 

Hold down the Space key to propel the owl upwards.

Release the Space key to fall downwards.
